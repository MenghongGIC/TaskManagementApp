# Task Management App

A simple JavaFX desktop application for managing tasks, built with Maven.

This project serves as a starting point for a full-featured task management system with FXML-based UI, controller separation, and potential database integration (e.g., SQL Server via JDBC).

# Command
`mvn clean` - Deletes the target folder (removes old builds).
`mvn compile` - Compiles the source code and checks for errors.
`mvn javafx:run` - The main command to launch the application.
`mvn package` - Creates a JAR file of your project in the target folder.

One command: ` mvn clean compile javafx:run `
